Chrome下载参数

发表于 2024-01-10

DEMO

https://www.google.cn/chrome/?system=true&standalone=1&platform=win64&extra=stablechannel

参数

system

值 含义
true 为所有用户安装
false 为当前安装

standalone

值 含义
1 本地安装
0 在线安装

platform

值 含义
win Windows 32位
win64 Windows 64位
win49 Windows XP/Vista
mac macOS 10.15 或更高版本
mac65 macOS 10.9
mac49 macOS 10.6 - 10.12
linux Linux

extra

值 含义
stablechannel 正式版
betachannel 尝鲜者版
devchannel 开发者版
canarychannel 探索者版

statcb

值 含义
0
1

defaultbrowser

值 含义
0 非默认浏览器
1 默认浏览器

installdataindex

值 含义
empty 非默认浏览器
defaultbrowser 默认浏览器
# 浏览器 # 工具
密钥管理服务 (KMS) 客户端激活和产品密钥
