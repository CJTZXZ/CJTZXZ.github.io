Windows安装

发表于 2024-01-11 更新于 2024-01-12

需要准备的材料：电脑，U盘（提前准备好安装材料）

U盘的准备

确保U盘没有重要数据，U盘的准备需要格式化U盘

U盘的准备有很多种方法，这里列举三种方法

Ventoy（进阶推荐）

官网: https://www.ventoy.net/cn/

  1. 在下载选项卡里下载ventoy-XXX-windows.zip(下载后注意检查 SHA-256 校验值)

  2. 解压并打卡压缩包的Ventoy2Disk.exe

  3. 选择要制作的U盘并点击安装

  4. 把系统的ISO镜像文件或 Windows PE 的ISO镜像文件放入U盘里

Windows PE （易用推荐）

以优启通为例

官网: https://www.itsk.com/eu

  1. 下载EasyU_XXX.7z

  2. 解压并打开压缩包的Ventoy2Disk.exe

  3. 选择U盘模式，选择U盘，点击全新制作

  4. 把系统的ISO镜像文件放入U盘里

PS: 选择生成ISO可以生成由 Ventoy 启动的ISO文件

微软官方（小白推荐）

官网: Windows 10 Windows 11

微软官方工具有详细说明，根据提示操作即可

  2. 把第一个启动项设置为U盘，设置适合自己的启动模式（UEFI和Legacy），建议新电脑设置为UEFI

  3. 不同的U盘情况会有不同的界面

    1. 微软官方工具制作的U盘会直接进入安装界面，根据自己需求按提示操作

    2. Ventoy 会进入选择界面，选择系统镜像同微软官方工具制作的U盘，选择PE工具生成的ISO文件同PE工具制作的U盘

    3. PE工具制作的U盘会进入系统选择界面，根据需要安装的操作系统，选择PE的系统版本

    如果不能进入页面并提示包含有safety，进入BIOS，将安全启动(Secure Boot)设置为disabled，安全启动(Secure Boot)一般在Boot标签页内

  4. 在PE内打开系统ISO镜像文件，即可安装操作系统，根据自己需求按提示操作，注意安装位置的选择

  5. 系统安装过程可能会有多次重启
